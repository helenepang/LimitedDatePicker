# LimitedDatePicker  
  

##一款Android日历控件，支持公历、农历转换  
  
   

**有时候需求是这样的，从日历中选择日期，但要指定可选的日期区间，纵观github上并没有这种源码，只有全套的日历，所以就花了点时间改写了一套，也供其他开发者参考。**
